Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B(não semântica).

Nesta frase temos um termo em destaque usando a tag STRONG(semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I(não semântica).

Nesta frase temos um termo em ênfase usando a tag EM(semântica).

Usar as tags STRONG para negrito e EM para itálico, pois elas são mais modernas e semânticas.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.
Importante: a tag BIG não deve ser mais utilizada por ser obsoleta.

Texto Excluído

Podemos marcar um texto como excluído utilizando a tag DEL.

Texto Inserido

Podemos marcar um texto como inserido(sublinhado) utilizando a tag INS.

Texto Sobrescrito e Subescrito

Podemos marcar um texto como sobrescrito utilizando a tag SUP.
Ex: x2
Podemos marca-lo como subscrito utilizando a tag SUB.
Ex: H2O.
