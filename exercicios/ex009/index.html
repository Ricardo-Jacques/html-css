Como Criar Listas

Lista OL

Lista ordenada.
Ex:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Almoçar
  6. Fazer exercícios

Lista UL

Lista Não Ordenada.
Ex:

  • Pão
  • Leite
  • Café
  • Queijo
  • Presunto
  • Banana

Lista DL

Lista de definição.
Ex:

HTML
Linguagem de marcação utilizada para criar o conteúdo de um site.
CSS
Linguagem de marcação utilizada para estilizar o conteúdo de um site.
JavaScript
Linguagem de programação utilizada para criar interação dentro de um site.
