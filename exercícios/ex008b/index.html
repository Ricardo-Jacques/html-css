Tags CODE e PRE

Podemos usar a tag CODE para tornar o texto mono-espaçado, como um código. E a tag PRE serve para manter a sua formatação.
Ex: 

    
        EscrevaL("Digite um número:")
        Leia(N1)
        EscrevaL("Digite outro número:")
        Leia(N2)
        Escreva("A soma entre estes dois números é"), (Soma))
